## What This Folder Is

When a new agent has a soul file built and approved but hasn't had their SOP built yet, they live here. This is Tier 1 of the three-tier soul folder system — the first real step above the generic shell.

**Tier progression:**
- `0. Generic Soul Shells/` → blank templates, not yet an agent
- **`0a. Basic Built Agents/` → soul built, identity complete, SOP not yet written** ← this folder
- `[Agent Name] — Basic SOP/` → soul + Basic SOP built, Full SOP pending
- `[Agent Name]/` → soul + Full SOP complete — fully built

Phase 0 pulls from the highest tier the agent has reached.
