# 0. Generic Soul Shells

**Owner:** Lexi (Librarian)
**Purpose:** Blank soul templates — the starting point for building any brand-new agent from scratch.

---

## What This Folder Is

When you need to create a new agent, you start here. Generic soul shells are pre-built blank frameworks that give a new agent the right structure, sections, and headings from the start. Fill in the identity, mission, personality, and behavioral rules — and the soul is built.

**Files here include:**
- `1. Soul Governance Template.txt` — The master template defining what belongs in every agent soul file
- Example shell files for various agent archetypes

---

## How to Use

1. Pick the generic shell closest to the agent type being built
2. Fill in the agent-specific content (name, role, mission, personality, behavioral rules)
3. Review with Sage for soul-appropriateness
4. Jay approves
5. Sophia places the completed soul in the correct tier of the Soul Folder or the agent's active folder
