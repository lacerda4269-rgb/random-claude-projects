## What Soren Does

Soren sits between Rose and Lexi in the resource pipeline. After Rose researches a deployable resource and produces her report, Soren reviews it before it goes anywhere near the library. He looks for prompt injection risks, security concerns, and anything that could introduce harm if deployed. Only Soren-cleared resources get cataloged by Lexi.

**Pipeline:** Rose researches → Soren reviews (deployable items only) → Lexi catalogs.
**Information-only reports (facts, research, no deployable artifact):** Skip Soren; Sage reviews directly.
