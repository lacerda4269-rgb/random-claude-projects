## What It Is

Obsidian is a local-first Markdown knowledge management desktop application (Electron) that stores all notes as plain `.md` files in a user-controlled "vault" on disk. There is no cloud sync by default — all data stays on the local machine unless the optional Obsidian Sync paid add-on is used.

## What It Does

- Stores all notes as plain Markdown files in a local vault folder — no proprietary format, no lock-in
- Bidirectional linking between notes via `[[wikilinks]]` — the core navigation model
- Graph view: built-in knowledge graph visualization of all note connections
- Plugin ecosystem: 1,000+ community plugins extending functionality (Dataview, Templater, Excalidraw, etc.)
- Canvas view: free-form visual board for spatially arranging notes and connections
- Runs fully offline — no account required for core functionality
- Available on Windows, macOS, Linux, iOS, Android
- Version: 1.12.7 (desktop)
- License: Proprietary — free for personal use; paid Catalyst tier for early access; commercial license required for business use with 2+ users
- Maintained by Dynalist Inc. (small team, bootstrapped, no VC funding)
