## What It Is
A curated list repository aggregating Claude Code resources — skills, agents, plugins, hooks, slash-commands, agent orchestrators, applications, and configurations — organized into a comprehensive reference index for the Claude Code ecosystem.

Aggregates and categorizes the Claude Code community's output across: Agent Skills, Workflows & Knowledge Guides, Tooling (IDE integrations, usage monitors, orchestrators), Status Lines, Hooks (lifecycle automation), Slash-Commands, CLAUDE.md configurations (language and domain-specific), Alternative Clients, and Official Anthropic Documentation.
