## What It Is
A comprehensive community-maintained reference guide covering best practices, patterns, and workflows for Claude Code development — described as "practice makes claude perfect."

## What It Does
The repository documents Claude Code's core concepts (Subagents, Commands, Skills, Workflows & Hooks, MCP Servers, Settings, Memory via CLAUDE.md files) with 86+ tips and tricks across prompting, planning, agents, skills, and debugging. It includes 8+ documented workflows from industry leaders, an orchestration pattern library (command → agent → skill sequences), and emerging features documentation covering Auto Mode, Channels (Telegram/Discord push events), multi-agent PR review, scheduled tasks, and voice dictation. 56.9k stars, actively maintained (updated 2026-06-08).
