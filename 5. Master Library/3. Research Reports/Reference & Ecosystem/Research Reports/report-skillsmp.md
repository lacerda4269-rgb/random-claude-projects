## What It Is
SkillsMP is a web marketplace for Claude Code skills — a community-maintained platform where developers publish, discover, and download skills for Claude Code. Functions as a browse-and-discover catalog for Claude Code behavioral extensions. No official Anthropic affiliation.

## What It Does
Aggregates Claude Code skills submitted by community contributors. Categories cover coding, testing, documentation, analysis, and general workflow patterns. Skills can be browsed by category, previewed, and downloaded for installation into `.claude/` project folders. The site also accepts community skill submissions. No authentication required to browse.

## Relevance to This Project
Discovery resource for Cosmo and Agent Onboarding skill sourcing. When the team needs a skill, SkillsMP is one of three places to check before building (alongside the official Anthropic catalog and claude-skills repo). Paired with claudemarketplaces.com — similar discovery purpose, different community.
