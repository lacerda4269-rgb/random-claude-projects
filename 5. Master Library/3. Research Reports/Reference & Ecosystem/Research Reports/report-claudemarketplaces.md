## What It Is
ClaudeMarketplaces.com is a community web marketplace for Claude Code resources — covering both skills and MCP servers. Aggregates community-built resources with listing pages, installation instructions, and capability descriptions. No official Anthropic affiliation. Reviewed alongside SkillsMP (Items 6+7) — similar purpose, different community and content range.

## What It Does
Aggregates Claude Code skills and MCP servers submitted by community contributors. MCP server listings include installation instructions, configuration notes, tool capability lists, and links to source repositories. Skills section operates similarly to SkillsMP. Browsable by category. Community-maintained with no centralized security vetting process.

## Relevance to This Project
Discovery resource for both skill sourcing and MCP awareness. MCP listings specifically are the highest-risk content category — MCP servers run with significant permission scope on the development environment. Use for awareness and discovery only; never install an MCP from this site without full Soren review. Paired with SkillsMP for pre-build skill discovery.
