## What It Is
A comprehensive performance optimization system for Claude Code — described as an Anthropic hackathon winner — bundling 28 specialized agents, 125+ skills, 60 commands, hooks, rules, and MCP configurations into a single installable package.

## What It Does
Provides a full-stack Claude Code enhancement layer: token optimization and model selection strategies, memory persistence across sessions, continuous learning that extracts patterns from development sessions, verification loops for code quality, multi-language support (TypeScript, Python, Go, Java, Kotlin, C++, Rust, PHP, Swift, Perl), security scanning via AgentShield integration, and multi-agent orchestration capabilities. Now supports Claude Code, Codex, OpenCode, Cursor, Gemini, Kiro, and more. Available as a Claude Code plugin with selective language-specific installation. Requires Claude Code CLI v2.1.0+. 210k stars, 32.3k forks, 2,054 commits (updated 2026-06-08). **Repo renamed from `everything-claude-code` to `ECC` — update any saved bookmarks.**

## Relevance to This Project
High relevance as a reference and parts source. The 125+ skills and 28 agents represent a massive catalog to mine for patterns, approaches, and reusable components — particularly around multi-agent orchestration, memory persistence, and verification loops, all of which are active concerns for this project. This is not something to install wholesale; it is a library to study and selectively pull from.
