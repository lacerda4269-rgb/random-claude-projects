## C&P Summary

Nanobot is an ultra-lightweight personal AI assistant runtime from HKUDS (same organization as LightRAG and RAG-Anything) with 99% fewer lines of code than typical predecessors. It supports multi-channel messaging (Telegram, Discord, WhatsApp, WeChat, Feishu), MCP integration, web search, and persistent memory. 43.9k stars, MIT licensed, v0.2.1 (June 1, 2026), 17 releases (updated 2026-06-08). Relevant to this team as a reference design for lightweight agent runtimes and as a potential channel for agent notification workflows — trading system alerts via Discord or Telegram are a plausible future application. Awareness item at Tier 3.
