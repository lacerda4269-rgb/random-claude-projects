## What It Is
A VS Code extension that renders AI coding agents as animated pixel art characters in a tiny virtual office. Characters animate based on agent activity — typing, reading, waiting.

## What It Does
- Animated pixel art agents visible in VS Code while agents work
- Activity-aware animations (different behaviors when typing vs. idle)
- Customizable office layouts
- v1.3.0 (latest April 14, 2026), 8.1k stars, 5 releases (updated 2026-06-08)
