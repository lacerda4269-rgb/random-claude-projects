**Status key:** Built ✅ | Pending ⏳ | Archived 🗄️
**Priority tiers:** Tier 1 = core/critical | Tier 2 = high value, phase-dependent | Tier 3 = useful, not urgent

**Hybrid items:** Filed under primary type. Secondary types shown in the Type column. One place per item — always.
