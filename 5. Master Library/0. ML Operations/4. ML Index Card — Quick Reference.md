> **One source, not two:** this roll-up is **not** a second install catalog. Actual installed resources live in the ML catalog (`3. Research Reports/0. Resources Installed/` + the Full Database). Do **not** hand-fill this with install rows — entries are added only through the pipeline below, in lockstep with Lexi's catalog, so the two never drift.

- **Universal Research Report:** report produced → reviewed for universal value → placed in ML `3. Research Reports/Universal Research Reports/` → Sophia logs here.
- **Paired with Lexi:** when Lexi updates her catalog from a Cosmo build, this roll-up updates in the same pass. The two always move together.

**Skills** — *on-demand workflows (slash command / context trigger)* — *(no entries yet — first entry when Cosmo completes and Lexi catalogs the first skill build)*
