# 2. Agents - Employees Descriptions

**Owner:** Lexi (Librarian) maintains; Sage (CEO) approves
**Purpose:** Short role and purpose descriptions for every agent on the team — used when briefing new Claude sessions on who does what.

---

## What This Folder Is

When a new Claude session starts, it doesn't automatically know who each agent is. The description files here provide a compact, session-ready summary of each agent's role, responsibilities, and key behaviors. One file per agent. Load the relevant ones at session start to bring the team into context quickly.

**Files here:**
One `.md` file per agent, named `[agent-name]-description.md`. Each file contains:
- Agent name and role
- Core responsibilities
- What they own and what they defer
- Key behavioral traits

To author a new specialty agent description, copy `_agent-description-template.md` (the blank authoring template in this folder) and fill it in.

---

Standing agents (Cosmo, Lexi, Rose, Soren, Sophia, Sage) + all specialty agents (Feynman, Nick, Pete, Todd, Vicky, and any new agents built during this project).
