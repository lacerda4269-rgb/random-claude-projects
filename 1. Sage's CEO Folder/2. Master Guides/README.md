# 2. Master Guides

**Owner:** Sage (CEO)
**Purpose:** The governance bible for every project — Sage's operating rules, Jay's companion version, and the Hard Rules master list.

---

## What This Folder Is

The Master Guides define how Sage runs every project. They cover escalation, session management, agent coordination, involvement scales, and every behavioral rule that applies across all work. The Hard Rules are a distillation — the rules that never bend.

**Files here:**
- `Master Guide - Sage's Version (SOP).md` — Full operational guide written for Sage. This is the authoritative source.
- `Master Guide - Jay's Version (SOP).md` — Jay's companion version. Same rules, written in plain language for Jay's reference and review.
- `Hard Rules — Master List.md` — All Hard Rules extracted and numbered. Quick-reference for the rules that override everything else.
