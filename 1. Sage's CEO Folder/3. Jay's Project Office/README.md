# 3. Jay's Project Office

**Owner:** Jay (Chairperson) — drops content here; Sage monitors
**Purpose:** Jay's handoff zone — documents, notes, context, and direction that Jay places here for the team to pick up.

---

## What This Folder Is

When Jay has something to hand off to the team — a document to review, context for a decision, raw notes, a reference file — it goes here. This folder is Jay's drop zone. Sage checks it at session start and routes anything that needs action.

**What Jay puts here:**
- Context documents for new phases or decisions
- Reference materials for the team to use
- Raw notes Jay wants captured and organized
- External files relevant to the project

**What happens to inputs:**
- Sage reviews and routes: some items get processed immediately, some are cataloged, some prompt questions back to Jay
- Items don't pile up here permanently — they get picked up and moved to their appropriate destination
