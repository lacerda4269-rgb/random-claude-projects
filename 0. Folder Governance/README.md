## What This Folder Is

This folder holds the structural governance documents for this version. It is the written and visual record of every folder and file decision made for the active version.

When the version advances, all three documents in this folder update to reflect the new state. They track the active version only — prior versions are archived separately.

---

## What Lives Here

| File | Description |
|------|-------------|
| `Folder & File Reference.md` | The written reference for every folder and file in this version. Tracks walk decisions, ownership, and ripple estimates. Living document — updated at each walk stop and after any structural change. |
| `Folder & File Tree.md` | The plain-text directory tree of the full version structure — every folder and file in walk order. Updates in parallel with the Reference and the Visual TreeView. |
| `Visual TreeView.md` | Jay's at-a-glance visual (Mermaid) map of the version structure. Updates in parallel with the Folder & File Reference. |

---

## How to Use

- **When navigating:** Check the Visual TreeView for a quick orientation.
- **When making structural decisions:** Log them in the Folder & File Reference — name, location, description, owner, and ripple estimate.
- **When advancing the version:** Update both files to reflect the new state before closing out the old version.
