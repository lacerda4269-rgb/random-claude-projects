## What This Is

## How to Start a New Project

1. Copy this entire MUIP (Master Universal Initial Package) folder to your new project location
2. Rename `4. Named Project - Live Folder/` to `4. [Project Name] - Live Folder/`
3. Open `Getting-Started.md` — it walks through every Phase 0 step
4. Sage and Sophia execute Phase 0

> **Agent Teams is required on every project.** Section 3a of Getting-Started.md covers the setup: wire Sophia into `.claude/agents/` at Phase 0 before any sessions begin. P/M Lead wiring follows the same section — conditional on whether parallel runs are needed.
