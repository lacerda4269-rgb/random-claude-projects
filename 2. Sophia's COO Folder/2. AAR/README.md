## What This Folder Is

This is Sophia's project-close deliverable folder. The After-Action Review (AAR) is the final retrospective on a completed project — what worked, what didn't, what gets promoted to the global lessons system, and what informs the setup of the next project.

The AAR is produced once per true project at close. It does not apply to mini-tasks, side projects, or mid-project checkpoints.
