## What This Folder Is

This is Sophia's working layer — the files she touches every session. Four core files are deployed here from `6. Blank Universal Templates/` during Phase 0 and stay active for the entire project.

**Files here (after Phase 0 setup):**
- `Gotcha.md` — Project watch list. Flags, known risks, things that will bite you if forgotten. Sophia adds entries as they emerge.
- `Project Resources Log.md` — Per-project tracker for all resources deployed (Skills, Hooks, CLIs, MCPs, Plugins). Every resource that goes live is logged here.
- `sophia-pending-changes.md` — Sophia's active queue of changes that need to happen. Sophia works through this list every session.
- `sophia-completed-changes.md` — Archive of completed changes. Closed items move here from pending.

**Blank version:** Empty at baseline. Phase 0 deploys all four operational files from `6. Blank Universal Templates/`.
